# Tobiuo Linux

**Tobiuo Linux**は、Debianをベースにした新しいLinuxディストリビューションです。ユーザーフレンドリーかつ軽量な環境を提供し、初心者から上級者まで幅広いユーザーに適しています。

> [!WARNING]
> このプロジェクトはまだ開発中であり、実用には適していません。

## 特徴

- [ ] KDE Plasmaデスクトップを採用し、かつ軽量な環境を提供
- [ ] 最新のソフトウェアパッケージを提供
- [ ] ユーザーフレンドリーなインストーラーと設定ツール
- [ ] 独自アプリケーションの開発と提供

## 技術スタック

- **ベース**: Debian Trixie
- **デスクトップ環境**: KDE Plasma
- **パッケージ管理**: APT
- **ビルド**: Debos
